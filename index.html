HR Dashboard

Internal Asset Management System

A full-stack Django web application for managing company assets, employee profiles, and asset request workflows — built for HR teams and staff members.

View Source Code Read README
Python 3.14 Django 6.0 SQLite Playfair Display Font Awesome 6 HTML5 / CSS3 JavaScript

Key Features

Everything built and tested in this project

Employee Directory

View, update, and delete all registered employees with role-based access control (HR vs Staff).

Asset Management

Add, edit, delete, and view company assets with images, descriptions, and live stock status badges.

Server-Side Search

Search assets across all database pages — not just the current page — with pagination preserved.

Request Workflow

Staff submit asset requests; HR approves or rejects with a reason. Stock count auto-updates on approval.

Inventory & History

Full inventory view with availability status and complete request history for audit trails.

Role-Based Auth

HR users access the full management dashboard; staff see only their own data and requests.

User Roles

Two distinct dashboards with separate permissions

HR Manager HR

  • View and manage all employees
  • Add, edit, delete company assets
  • Approve or reject asset requests
  • Enter rejection reasons for declined requests
  • View full request history and inventory
  • See live stat cards (employees, assets, requests)

Staff Member STAFF

  • View personal employee profile
  • Browse available asset catalogue
  • Submit asset borrow requests
  • Track own request status (Pending/Approved/Rejected)
  • View personal request history
  • See company inventory availability
